# AI-PAPS: Scrum Documentation

## Обзор проекта

**AI-PAPS** (Artificial Intelligence - Plant Analysis and Protection System) — интеллектуальная веб-платформа для автоматической диагностики болезней и вредителей сельскохозяйственных растений с использованием технологий компьютерного зрения и машинного обучения.

## Структура документации

```
scrum/
├── README.md                    # Обзор проекта (этот файл)
├── PRODUCT-BACKLOG.md           # Полный бэклог продукта
├── MODULES.md                   # Описание модулей системы
├── DATABASE.md                  # Схема базы данных
├── API.md                       # API endpoints документация
├── sprints/
│   ├── SPRINT-1.md              # Инфраструктура и аутентификация
│   ├── SPRINT-2.md              # Справочники и каталоги
│   ├── SPRINT-3.md              # Управление датасетами
│   ├── SPRINT-4.md              # Нейронные сети и модели
│   ├── SPRINT-5.md              # Обучение моделей

```

## Команда проекта

| Роль | Ответственность |
|------|-----------------|
| Product Owner | Определение требований, приоритизация бэклога |
| Scrum Master | Организация процессов, устранение препятствий |
| Frontend Developer | Реализация UI на React/Next.js |
| Backend Developer | Реализация API на C#/ASP.NET Core |
| ML Engineer | Разработка и обучение моделей PyTorch |
| DevOps Engineer | Инфраструктура, CI/CD, контейнеризация |
| QA Engineer | Тестирование, обеспечение качества |
